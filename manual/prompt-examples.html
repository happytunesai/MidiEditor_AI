🔄 MIDI Conversion & Cleanup

Fix, reorganize, and clean up MIDI files.

Channel remap
Move all notes from Channel 0 to Channel 2, and reassign the drums from Channel 5 to Channel 9
Track split
This file has all notes on a single track. Split them into separate tracks by channel — one track per channel, named by their GM instrument.
Cleanup
Remove all notes shorter than 10 ticks (ghost notes) and delete any empty tracks
Range limit
Clamp all notes to the range C2–C7. Transpose any notes outside this range by octaves until they fit.

💡 Prompt Writing Tips

  • Be specific: Include tempo, key, length, and instrument names for best results
  • Use Agent mode for multi-step tasks (composing, arranging, orchestrating)
  • Use Simple mode for quick single-step edits (transpose, velocity, delete)
  • Reference tracks by name or number: “Track 1” or “the Piano track”
  • Iterate: Start with a basic composition, then refine with follow-up prompts
  • Use the context range setting to control how much of the file the AI sees
  • Check the API log if something doesn’t work as expected
